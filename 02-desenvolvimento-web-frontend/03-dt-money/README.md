# 💸 DT Money

![banner](src/assets/banner.png)

Sistema de controle financeiro pessoal desenvolvido para gerenciar entradas, saídas e visualizar o saldo total de forma simples e intuitiva.
---

## 📊 Visão Geral

O **DT Money** é uma aplicação web para controle financeiro que permite:

- Registrar novas transações
- Visualizar entradas e saídas
- Acompanhar o saldo total atualizado
- Buscar transações específicas
- Organizar movimentações por categoria
- Navegar entre páginas de registros
